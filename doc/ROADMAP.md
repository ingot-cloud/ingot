# Roadmap

## 平台功能
- [x] 用户管理
    - [x] 添加用户
    - [x] 编辑用户，包括组织编辑
- [ ] 组织管理
    - [x] 创建组织，需要创建默认信息，管理后台角色和app角色(默认租户的组织类型角色)，初始化部门，初始化管理员用户，初始化菜单
    - [x] 编辑组织
    - [ ] 组织初始化管理
        - 配置默认初始化菜单
        - 配置默认初始化部门
        - 配置默认初始化角色
        - 修改任意默认值后，需要更新所有租户对应的默认配置
    - [ ] 组织功能管理
        - 添加、删除平台提供的功能 
- [x] 权限增加权限类型
- [ ] 平台基础管理，功能需要梳理，接口需要整理
- [ ] ID管理，biz_leaf_alloc表的管理
